# Key-Vora-Cloud
Key Vora Cloud es un gestor de contraseñas creado para generar contraseñas totalmente seguras y poder almacenarlas en ese mismo momento
Las contraseñas se almacenan en la nube en una base de datos privada, esto lo hacemos por si llegas a formatear tu computador con tu usuario y clave maestra puedes volver a acceder a todas tus contraseñas guardadas pero esto es peligroso?
No, nuestra base de datos aparte de ser privada esta totalmente cifrada es decir tu clave maestra como tus contraseñas generadas y almacenadas están totalmente cifradas y ni si quiera nosotros podemos acceder a tus datos la único información que se puede extraer es un hash bastante largo el cual sin nuestros métodos es imposible de descubrir los datos reales.
asi que anímate a probarlo al ser la versión 1.0 puedes tener errores cualquier sugerencia de cosas que se puedan agregar o bugs favor de reportarlos por GitHub o un mensaje a mi discord: "muck0178"
